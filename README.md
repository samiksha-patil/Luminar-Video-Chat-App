## Features

- **Create Meeting**
- **Join Meeting**
- **Live Video Call**
- **Real Time Chat**
- **View Participants**
- **Mute button to turn off the audio**
- **Video button to turn off the video**

## Technology Stack

- NodeJs
- WebRTC
- Express Server
- Peerjs Library
- Socket.io
- ejs
- HTML5
- CSS
- jQuery
- Javascript
