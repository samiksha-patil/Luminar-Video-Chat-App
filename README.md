## Features

- **Create Meeting**
- **Join Meeting**
- **Live Video Call**
- **Real Time Chat**
- **View Participants**
- **Mute button to turn off the audio**
- **Video button to turn off the video**

- NodeJs
- WebRTC
- Express Server

- ejs
- HTML5
- CSS
- jQuery
- Javascript
